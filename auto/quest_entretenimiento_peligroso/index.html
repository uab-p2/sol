    Entretenimiento peligroso¶

    El camino será largo. ¿Por qué no fabricarte algo de entretenimiento? Te guiamos para montar un juego peligroso: el ahorcado.

    Objetivo
    • Diseña y dibuja las entidades, datos e interacciones que describen el juego.
    • Nombra las acciones que tienen lugar en cada interacción.
    • Implementa el juego comenzando por las funciones más sencillas.

    Dibujo¶

    Antes de programar necesitamos saber qué queremos programar. Coge un papel. Ahora, coge un boli (o un lápiz). Tú y yo sabemos cómo es el juego del ahorcado, pero ¿sabrías explicarlo únicamente con dibujos?

    • ¿Qué entidades participan?
    • ¿Cómo se relacionan?
    • ¿Qué datos manejan?

    Te proponemos que lo expliques utilizando diagramas de secuencia, de estado y de flujo como los de Diagramas y programación (WIP).

    Intercambio de idiomas¶

    Elegir las palabras correctas nos ayuda a pensar mejor, producir un código más expresivo y barato de mantener. Repasa los diagramas del punto anterior y encuentra la palabra idónea. Usa:

    • Nombres: para las entidades y datos.
    • Verbos: para las acciones.

    Mecanografía¶

    ¿Ya lo tienes todo preparado para comenzar a teclear? Te proponemos los siguientes pasos:

    1. Abre quest/entretenimiento_peligroso y comprueba que compila sin problema.

    2. Crea el fichero hangman.h en la raíz del quest. Crea aquí las declaraciones de funciones y structs con la nomenclatura del punto anterior antes de ponerte con la implementación.

    3. Crea el fichero hangman.cpp, también en la raíz del quest. Una a una, implementa las funciones de hangman.h de la siguiente manera:

      1. Elige la función más sencilla pendiente de implementar.
      2. Crea una primera versión de esa función en hangman.cpp.
      3. Prueba esa función desde test.cpp hasta que funcione. No olvides el #include "hangman.h".

    4. En el main() de main.cpp, implementa la lógica del juego usando únicamente las funciones que hayas implementado en hangman.cpp.

    Quizá te pueda ayudar:

    • Silicio y titanio
    • Diagramas y programación (WIP)

    Literatura¶

    Recuerda que puedes leer de teclado y escribir por consola con 

    #include <iostream>
#include <string>

int main(int argc, char* argv[]) {
   std::cout << "Nombre: " << std::endl;
   std::string nombre;
   std::cin >> nombre;
}

    Por si quieres hacer el juego un poco más interesante, hemos incluido una carpeta words/ con algunas de las palabras más frecuentes de cada idioma. Lo veremos en detalle más adelante, pero puedes usar el siguiente código para leer palabras de los ficheros en words/:

    #include <fstream>
#include <iostream>
#include <string>

int main(int argc, char* argv[]) {
   std::ifstream f("words/es_10k.txt");
   std::string word;
   while (f >> word) {
      std::cout << "I just read the word '" << word << "'" << std::endl;
   }
}
