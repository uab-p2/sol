# Solución: Silicio y titanio

Instalación de la plataforma de desarrollo y ejecución de un proyecto 
de código ya dado. No se les pide programar nada.

Probablemente más útil si se les pide intentar la instalación en casa
y dedicar la sesión a solucionar problemas de instalación 
(y plantear algún ejercicio ligero al resto).

Duración: 30 minutos.
