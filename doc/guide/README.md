# Guía de estudio

Campañas de quests y documentos de referencia.

## Compilación

- `make`: compilado estático
- `make serve`: servicio dinámico

### Requisitos previos

sudo apt install python3 python3-pip make 

pip install mkdocs mkdocs-material pymdown-extensions mkdocs-minify-plugin mkdocs-macros-plugin

## Añadir contenido

- Modificar el índice en `mkdocs.yml'.
- Añadir contenido Markdown en `sections/`
